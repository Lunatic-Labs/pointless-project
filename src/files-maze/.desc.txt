The instruction set for this maze is `uuurddruur`. However, this is too many instructions for the robot.

Luckily it remembers a certain compression algorithm called <a href="https://en.wikipedia.org/wiki/Run-length_encoding">run-length encoding</a>
and decides to try it out. The algorithm goes something like:

<ul>
<li> Do not modify single characters. </li>
<li> For every sequence of the same consecutive letter, compress it by using the letter followed by the <b>number</b> of consecutive letters.</li>
</ul>
