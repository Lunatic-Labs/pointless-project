The instruction set for this smaller maze is `uuurddruur`. However, for the larger labyrinth, there are too many moves for the robot to process—**speed and efficiency are the key to survival**.<br/>

Fortunately, the robot remembers a trick from its programming: <a href="https://en.wikipedia.org/wiki/Run-length_encoding">Run-Length Encoding</a>. This algorithm compresses instructions to make them shorter and faster. Here's how it works:

<div>
  <ul>
    <li> Do not modify single characters. </li>
    <li> For every sequence of the same consecutive letter, compress it by using the letter <br /> followed by the <b>number</b> of consecutive letters. </li>
  </ul>
</div>
